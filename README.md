# TP Carrefour (mulitâches)

## 1. Pour compiler

### 1.Cloner le dépot git
```shell
$ git clone https://github.com/darkfrancky/tpMult
```

### 2.Installer les librairies
```shell
$ apt-get install libtcl8.5-dev libncurses5-dev
```

### 3.Construire à partir des sources
```shell
$ cd tpMult
$ make
```
